Pentaho data integration(kettle) 在Mac上启动失败

2019-07-01

环境

  • MacOS Mojave (10.14.1)
  • Pentaho Data Integration 8.2
  • Java 8

问题描述

使用的解压包，解压之后，放到了Application中，然后打开Data Integration。

第一次打开，提示安全问题，需要授权，可在系统偏好设置>安全性与隐私中设置。

第二次打开，没有任何反应。

解决方式

从Terminal启动

从terminal进入到解压之后的目录data-integration中，找到脚本spoon.sh，然后执行它：./spoon.sh，启动成功。

从Launchpad启动

想要从Launchpad双击图标启动，就得解决权限的问题。

随着Mac系统版本升级，Mac的权限控制越来越严。在网上找到这篇帖子，虽然系统版本和我的略有差别，但是原理是一样的。

在解压之后，把整个data-integration移动到/Applications目录下，然后执行：

sudo xattr -dr com.apple.quarantine /Applications/data-integration/Data\ Integration.app

执行成功之后，再双击图标就可以打开程序了。
